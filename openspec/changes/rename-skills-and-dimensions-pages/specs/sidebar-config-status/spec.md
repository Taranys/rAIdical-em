## MODIFIED Requirements

### Requirement: Configuration status indicators for Review Categories and Competencies
The sidebar SHALL display a green check icon for the "Review Categories" and "Competencies" items, matching the existing behavior previously attached to the "Skills" and "Dimensions" labels.

#### Scenario: Review Categories shows green check
- **WHEN** the sidebar is displayed
- **THEN** the "Review Categories" sidebar item displays a green check icon

#### Scenario: Competencies shows green check
- **WHEN** the sidebar is displayed
- **THEN** the "Competencies" sidebar item displays a green check icon
